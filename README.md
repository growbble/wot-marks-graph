# WoT Marks Graph

Мод для World of Tanks (Lesta), который отображает график прогресса отметок на стволе в реальном времени.

## Возможности

- 📊 График изменения процента отметки в реальном времени
- 🎯 Отображение текущего прогресса боёв
- 📈 Прогноз изменения отметки после боя
- 🖱️ Drag & Drop для перемещения виджета
- 🔒 Блокировка виджета в бою (опционально)
- 💾 Автосохранение истории боёв

## Установка

1. Скопируйте содержимое репозитория в папку `res_mods/<версия_игры>/scripts/client/gui/mods/`
2. Запустите игру

## Использование

- В ангаре отображается кнопка расширения + панель графика
- В бою график показывает прогресс отметки в реальном времени
- Перетаскивайте виджет зажав левую кнопку мыши

## Зависимости

- Python 3.8+
- World of Tanks (Lesta)
- WoT Development Framework
